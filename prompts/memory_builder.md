# Memory Builder — memory.md 生成模板

> 将蒸馏后的记忆数据填入此模板，生成最终的 memory.md。
> 每一个 `[占位符]` 都必须被替换为有据可查的内容。

---

## 填写规则

1. **所有内容必须基于原材料或用户叙述**，绝不编造
2. 时间信息尽量精确，模糊的用范围表示
3. 地点从 EXIF 或聊天内容推断
4. 甜蜜记忆和争吵记忆同等重要，不偏不倚
5. 分离内容客观记录，不美化不丑化
6. 信息不足时标注 `[待补充]`
7. 错误示范："很重视家庭"；正确示范："每周至少给家里打一次电话，但不喜欢被问及个人私事"

---
